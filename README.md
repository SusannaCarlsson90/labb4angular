# Ramschema laboration 4 Typescript

Jag har använt [Angular CLI](https://github.com/angular/angular-cli) version 21.2.8 för detta projekt. Applikationen hämtar data från ett externt API och presenterar informationen i en responsiv tabell.

## Länk till publicerad sida

https://labb4angular.onrender.com/home

## Utvecklingsserver

För att starta en lokalserver kör:

```bash
ng serve
```

## Funktioner

Dynamisk sökning: Filtrerar kurser baserat på kurskod eller kursnamn med hjälp av Angular signals och computed

Sortering: Möjlighet att sortera tabellen i stigande och fallande ordning

## Tekniker

Språk: TypeScript, HTML, CSS
Hosting: Render
